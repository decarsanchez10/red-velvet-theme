# Velvet Noir

A rich, romantic dark theme built around a hand-picked velvet color palette. Velvet Noir brings a warm, wine-toned atmosphere to your editor — perfect for late-night coding sessions.

![Velvet Noir Screenshot](./img.png)

## Color palette

| Color | Hex | Used for |
|---|---|---|
| Merlot | `#2C0415` | Editor background |
| Garnet | `#720008` | Keywords, accents |
| Crimson | `#9F305E` | Functions, highlights |
| Orchid | `#B175A5` | Strings, variables |
| Petal | `#EE9FB3` | Comments, subtle text |

## Features

- Deep Merlot background reduces eye strain in low light
- High-contrast syntax colors for clarity without harshness
- Cohesive UI theming across sidebar, tabs, and status bar
- Pairs beautifully with most icon theme packs

## Installation

1. Open the **Extensions** view in VS Code (`Ctrl+Shift+X`)
2. Search for **Velvet Noir**
3. Click **Install**
4. Open the Command Palette (`Ctrl+Shift+P`) → **Preferences: Color Theme** → select **Velvet Noir**
